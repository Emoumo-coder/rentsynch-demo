# RentSynch Africa Demo Dashboard

A responsive, accessible property management dashboard built with Next.js 14, TypeScript, and Redux Toolkit.

## Features

### Core Requirements
- **Login Screen**: Clean, accessible login interface with form validation
- **Dashboard Overview**: Summary cards with key metrics and statistics
- **Property List Table**: Interactive table showing all properties with status indicators
- **Recent Activity Section**: Timeline of recent activities and events
- **Rent Status Section**: Visual representation of rent collection status

### Bonus Features
- **Dark/Light Mode Toggle**: Full theme switching with system preference detection
- **Animations & Transitions**: Smooth animations using Framer Motion
- **Accessibility**: ARIA labels, keyboard navigation, screen reader support
- **Responsive Design**: Mobile-first approach with breakpoint optimization

### Technical Features
- **State Management**: Redux Toolkit with typed slices
- **Component Architecture**: Reusable, composable components
- **Type Safety**: Full TypeScript implementation
- **Performance**: Code splitting, optimized animations

## Getting Started

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/Emoumo-coder/rentsynch-demo
cd rentsynch-demo

# Install dependencies
npm install

# Run development server
npm run dev
